# Black Diamond Brawl

A Road Rash-style downhill snowboarding combat racer, built as a learning project.

Solo score-attack runs down a procedurally generated slope — dodge trees, rocks, and rival
riders, shove or weaponize your way past them, and rack up distance and trick points before
you wipe out.

Built with Phaser 3 + TypeScript, using a classic segment-based pseudo-3D renderer for the
behind-the-rider "road rushing at you" look (OutRun/Road Rash style).

## Status

Design in progress — no gameplay code yet.
